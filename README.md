
# Java To-Do List App 

This is a simple **console-based To-Do List application** written in Java for the SWE 202 - Software Verification and Validation course.

The app allows users to:
- Add new tasks with a title and description
- View all tasks
- Mark tasks as completed
- Delete tasks

---

## Project Structure

```
/src
 └── org
      └── example
           ├── Main.java
           └── classes
                └── Task.java
```

---

##  How to Run

1. Clone this repository:
   ```bash
   git clone https://github.com/rhmsarr/ToDoApp-SVV-HW1.git
   ```

2. Compile and run:
   ```bash
   cd your-repo-name
   javac -d . src/org/example/Main.java
   java org.example.Main
   ```
